技巧一：浮动

Box-1
Box-2
      
.trick-1 .container {
  width: 500px;
  height: 200px;
}

.trick-1 .container .box {
  width: 200px;
  height: 200px;
}

.trick-1 .container .box-1 {
  float: left;
  background-color: #00B7FF;
}

.trick-1 .container .box-2 {
  float: right;
  background-color: #7979DE;
}
      
    


<div class="container">
  <div class="box box-1">Box-1</div>
  <div class="box box-2">Box-2</div>
</div>

container 包裹要水平排列的两个元素，注意，浮动只适合一个父元素下包裹两个子元素的情况。

container 的高要和两个子元素的高保持一致，宽必须大于两个子元素宽度之和。两个子元素分别使用 float: left；float: right
